<div className="subheading">Design Tokens</div>
Design tokens are indivisible pieces of a design system such as colors, spacing, and typography. 
Using tokens allows designers and developers to classify these pieces by roles and usage, rather than their actual hard-coded values (such as hex values for color or pixel values for spacing). 
Using tokens also allows the use of similar labels for different tokens based on the theme in which they exist. 
This approach enables efficiency when updating colors within a theme, as well as any user interface built using design system to toggle easily between themes.
